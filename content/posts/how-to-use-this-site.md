There are three things on this site: posts, events, and links to a couple of external tools. Here's what each one is for.

## Posts

Posts are the main content — video companions, project write-ups, and station notes. If a video goes deeper than what fits on screen, the show notes live here. If a project involves enough decisions to be worth explaining, there's a write-up. If we spend enough time at a station to have opinions about it, we'll write those down too.

You can browse everything from the [Posts](/posts/) page in the nav.

## Events

The [Events](/events/) page lists railfan events we're planning to attend. If you'll be at the same event, come say hello. Events drop off the list automatically once they're over, so it only ever shows what's coming up.

## External Tools

Two tools worth keeping in a tab if you follow along with the channel:

- **[Station Board](https://station-board.pages.dev)** — a live arrivals board for any Amtrak station. Type in a station code and it shows you what's coming and how late it is. No account, no setup.
- **[Amtrak Tracker](https://asm.transitdocs.com/)** — a live map of every Amtrak train on the network. Useful for watching a train work its way toward you, or for following along when a video references a specific run.

We use both of these constantly to figure out when the trains are coming.
